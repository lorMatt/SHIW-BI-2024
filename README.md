# SHIW-BI-2024
This repo contains all the source codes and materials I was supposed to use in order to write the Inequality and Poverty report for the association [una Regione per Restare](https://www.instagram.com/regione_per_restare/). The project has been since terminated, so the interactive visualisations available on my [website](https://lormatt.github.io/posts/RxR%20Report%20inequality%20and%20poverty/) are the only result. The analysis was conducted in September 2024 and is based on the [Survey on Household Income and Wealth](https://www.bancaditalia.it/statistiche/tematiche/indagini-famiglie-imprese/bilanci-famiglie/distribuzione-microdati/index.html?com.dotmarketing.htmlpage.language=1) (Bank of Italy).
The repository contains the original data set, source code, cleaned datasets (in RDS format), exported visualisations and a .qmd version of the complete report.
## How to navigate the repository
The Report.qmd file should work immediately after loading the repository in an R working directory. I personally suggest that one starts exploring the results this way.
The source code and original data (in Stata .dta format) are also available in the repo. The two R scripts should be run in sequence: 'Data management' -> 'Data analysis'. The outputs of the Data management file, however, are the two .rds files already available in the repo. This way, if one only wants to investigate the actual analysis and has no interest in the data manipulation process, it is possible to dive right into the results.
A prototype of all data visualisation is exported in the 'img' folder. These images were mainly used as a way to show the rest of the organisation what kind of visualisation I had planned to employ. The final version of all plots can be found in 'Report.qmd' and in the final report (*forthcoming*).
